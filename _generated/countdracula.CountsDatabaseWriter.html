countdracula.CountsDatabaseWriter¶

class CountsDatabaseWriter(host='localhost', database='countdracula', username='cdadmin', pw='', logger=<logging.Logger instance at 0x028E2490>)¶

Bases: object

Class to handle write operations to the Counts database

Constructor; connects to the counts database with the given credentials.

  • host is the host where the counts database resides; e.g. ‘localhost’ or an ip address, like ‘172.0.0.1’
  • database is the database name; countdracula is the standard
  • username is a database user with write permissions; cdadmin is the standard.
  • password is the password for that the user.
  • logger is a logging.Logger instance for errors and informational messages.

insertMainlineCounts(countsList)¶

Inserts countsList into the counts_ml Table; see that documentation for more on what these variables actually represent.

countsList is a list of tuples, each tuple containing the following values;

  • count
  • starttime
  • period
  • vtype
  • onstreet
  • ondir
  • fromstreet
  • tostreet
  • refpos
  • sourcefile
  • project

Database errors will be ignored.

insertNodeStreets(nodeStreetList)¶

Inserts the given list of intersections into the node_streets Table.

The nodeStreetList is a list of tuples with the following values:

  • intersection_id - an integer id, should correspond with an int_id from nodes Table.
  • streetname - should correspond with a street_name or nospace_name from street_names Table.

Can be used with CountsWorkbookParser.readIntersectionIds()

insertNodes(nodeList)¶

Inserts the given list of intersection nodes into the nodes Table.

The nodeList is a list of tuples with the following values:

  • intersection_id - an integer id.
  • x - the x coordinate, such as the latitude
  • y - the y coordinate, such as the longitude
insertStreetNames(streetnameList)¶

Inserts the given list of streetnames into the street_names Table.

The streetnameList is a list of tuples with the following values:

  • street_name
  • nospace_name
  • short_name
  • suffix

Can be used with CountsWorkbookParser.readStreetNames()

insertTurnCounts(turnCountList)¶

Inserts the given list of turn counts into the counts_turns Table.

The turnList is a list of tuples with the following values:

  • count
  • starttime
  • period
  • vtype
  • fromstreet
  • fromdir
  • tostreet
  • todir
  • intstreet
  • intid
  • sourcefile
  • project

See counts_turns Table for the descriptions of the fields.
